Дополнительное описание находится в коде, в папке block-2

Приложение на angular разбито на комопоненты каждый из которых отвечает за свои задачи, код в стилях, шаблонах и тд разных
компонентов не пересекается, инкапсулируется внутри каждого компонента.
Компоненты делятся на:

app.component.css -> отвечает за стили html комопнента
app.component.html -> компонент(шаблон) html разметки
app.component.spec.ts -> тестирование
app.component.ts -> содержит код, методы, переменные, селектор и ссылки на одноименные компоненты стилей(.css) и шаблон(.html)
app.module.ts -> основной модуль приложения, здесь объявляются все созданные компоненты(.ts) и модули, модули angular,
                 сервисы, а также главный компонент приложения, обычно это AppComponent

запуск приложения в angular начинается с main.ts:
1. С помощью встроеной функции platformBrowserDynamic().bootstrapModule(AppModule) загружается и запускает основной модуль.
2. Модуль в свою очередь объявляет компоненты, импортирует модули, предоставляет сервисы и запускает основной компонент AppComponent.
3. Каждый компонент содержит селектор с помощью которого он помещатся в html. AppComponent добавлен в index.html,
   с которого начинается отрисовка в браузере.

Доп 2 способа задания селекторов:
  1. Значение можно взять в квадратные скобки: selector: '[app-comp]',
    и использовать селектор как атрибут, например:  <div app-comp></div>
  2. Перед значением ставится точка: selector: '.app-comp',
    и селектор используется как класс, например: <div class="app-comp"></div>
